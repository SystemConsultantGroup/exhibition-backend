== 공통 응답 형식

이 API는 엔드포인트 성격에 따라 아래 응답 형식을 사용합니다.

=== 목록 조회 응답 (페이지네이션)

목록 조회 API는 `items`, `page`, `pageSize`, `total`을 포함한 페이지네이션 구조를 사용합니다.

[source,json]
----
{
  "items": [
    {
      "...": "리소스 필드"
    }
  ],
  "page": 1,
  "pageSize": 20,
  "total": 120
}
----

=== 단건 조회 응답

단건 조회 API는 해당 리소스 객체를 그대로 반환합니다.

[source,json]
----
{
  "id": "uuid",
  "...": "리소스 필드"
}
----

=== 파일 다운로드 응답

파일 조회 API(`/media/{id}`)는 JSON이 아닌 바이너리 본문을 반환하며, `Content-Type`은 파일 형식에 따라 달라집니다.
